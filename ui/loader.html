Loader

Loader

Props

PropTypeDefaultDescription
typeLoaderTypedefaultThe visual style of the loader. Can be set via prop or boolean attribute.
sizeLoaderSizenullThe size of the loader. Can be set via prop or boolean attribute.

The supported loader types are imported from LOADER_TYPES, and typically include:

  • default

  • dotsBounce

  • dotsFade

  • dotsMove

  • dotsRotate

  • dotsScale

installation

import { FgLoader } from 'fadgram-vue'
